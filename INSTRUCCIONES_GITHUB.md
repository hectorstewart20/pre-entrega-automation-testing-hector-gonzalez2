Pequeñas instrucciones para subir el repo a GitHub.

1) Crear repo en GitHub: `pre-entrega-automation-testing-hector-gonzalez` (public)

2) En tu máquina, añadir remote (reemplaza la URL):
```bash
git remote add origin https://github.com/[tu-usuario]/pre-entrega-automation-testing-hector-gonzalez.git
```

3) Push:
```bash
git push -u origin master
```

Si te pide usuario/token, usa tu usuario y token personal.
